# Plasma-Widgets
A collection of custom and simple KDE Plasma widgets for enhanced desktop functionality and customization. 

## re.unam.ipaddress
A simple plasmoid widget that shown the public IP address in the panel. The URL for retrieve the public IP address can be modified easily. Also, it can retrieve the IP address each n-seconds.

## re.unam.ipaddress\_plasma6
A simple plasmoid widget that shown the public IP address in the panel. The URL for retrieve the public IP address can be modified easily. Also, it can retrieve the IP address each n-seconds. Plasma 6.

## re.unam.caldavtasks
A Plasma 6 widget to view and manage your VTODO tasks from any CalDAV server (Nextcloud, SOGo, Fastmail…). Browse tasks, add new ones, and mark them complete right from your panel.
